    Getting started

    1. Create MapTiler cloud account.
    2. Obtain the api key.

    3. Set the key for MapTiler in the application manifest (/app/src/main/AndroidManifest.xml)

      <meta-data android:name="com.maptiler.simplemap.mapTilerKey" android:value="your key" />
